# compas_cpp



## Installation

Stable releases can be installed from PyPI.

```bash
pip install compas_cpp
```

To install the latest version for development, do:

```bash
git clone https://github.com//compas_cpp.git
cd compas_cpp
pip install -e ".[dev]"
```
